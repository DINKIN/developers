Alpha Notes

{% include alpha-notice.html %}

To play with our new API during alpha, you should visit the alpha sign-up and register for an account in the alpha environment. A few notes about the alpha:

  • We are willing to make breaking changes during alpha.
  • We will wipe the database weekly. You will lose everything on your account, but you don't need to sign up again.
  • Some features may not work, and some features aren't even done yet.

We want your feedback! Submit issues on GitHub and join us on IRC in #linode on irc.oftc.net.
